# Vanta Motors

A polished Next.js 3D landing page built with React, Three.js, and Motion. This project showcases immersive scroll-driven branding and interactive visuals powered by React Three Fiber, Drei, and Motion.

## Demo
- A sleek homepage experience with animated stage transitions
- 3D backgrounds, particle effects, and smooth scroll interactions
- Designed as a modern showcase for a futuristic automotive brand

## Features
- Next.js 15 app router structure
- React 19 and TypeScript support
- 3D scene rendering with `three`, `@react-three/fiber`, and `@react-three/drei`
- Motion-based animation for polished UI transitions
- Modular component architecture in `app/components`

## Installation
```bash
npm install
```

## Development
```bash
npm run dev
```

Open `http://localhost:3000` after starting the development server.

## Production Build
```bash
npm run build
npm run start
```

## Project Structure
- `app/` — Next.js application files and components
- `app/components/` — reusable UI and 3D scene components
- `public/` — static assets used by the site
- `package.json` — dependencies and scripts
- `tsconfig.json` — TypeScript configuration

## Tech Stack
- Next.js
- React
- TypeScript
- Three.js
- React Three Fiber
- Drei
- Motion

- Ignore local build and dependency folders in Git: `.next/`, `node_modules/`
